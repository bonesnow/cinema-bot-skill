# 08｜飞书凭证安全配置

## 基本原则

App Secret 相当于应用密码。一旦出现在聊天记录、截图、日志或公开文件中，应当视为已经泄露。公开仓库只保留 `.env.example`，不会保存你的 App ID、App Secret 或 Verification Token。

请在飞书开放平台进入：

`应用 → 凭证与基础信息 → App Secret → 重置`

## 在 VPS 上安全写入

部署后执行：

```bash
cd cinema-bot-skill
./scripts/configure_feishu.sh
```

脚本会依次询问：

1. App ID
2. 新生成的 App Secret
3. Verification Token

Secret 和 Token 输入时不会显示在终端上。脚本会生成 `.env`，并将权限设置为：

```bash
chmod 600 .env
```

## 检查配置

```bash
docker compose build
docker compose run --rm cinema-bot python -m app.config_check
```

配置检查不会输出任何密钥内容。
